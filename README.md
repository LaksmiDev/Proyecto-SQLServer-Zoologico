# Proyecto SQL Server - Zoológico 

Este repositorio contiene el script SQL corregido de una base de datos relacional en **SQL Server** para un sistema de zoológico.

## Archivo principal

- `scripts/bd_zoologico.sql`

## Cómo ejecutarlo

1. Abrir **SQL Server Management Studio**.
2. Abrir el archivo `scripts/bd_zoologico.sql`.
3. Ejecutar el script completo en una instancia de SQL Server.
4. Verificar que se cree la base de datos `Zoologico`.

## Contenido

El script incluye:

- Creación de la base de datos `Zoologico`.
- Tablas relacionales con claves primarias y foráneas.
- Vistas.
- Índices.
- Datos de prueba.
- Procedimientos almacenados.
- Procedimientos transaccionales.
- Consultas paginadas.

## Nota

No se incluyen archivos `.bak`, `.mdf` ni `.ldf`, porque para GitHub es mejor subir el proyecto como código SQL.
